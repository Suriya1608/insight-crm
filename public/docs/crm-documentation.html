Zoom
API credential setup. Schedule or start instant Zoom meetings with leads. Test connection from admin.

Webhook integration to auto-capture leads from Facebook ad forms directly into the CRM.
🔗
External Lead API
CORS-enabled API endpoint for WordPress, landing pages, and third-party tools to push leads directly into the CRM.

TCN Softphone — Key Features

  • Embedded softphone iframe in the CRM interface
  • Per-user OAuth authentication (admin-linked)
  • Manual dial with operator control
  • Call hold, resume, and DTMF tone support
  • Incoming caller lookup (matches to CRM lead)
  • Automatic call log creation on call end
  • Call outcome recording via modal after each call
  • Agent status management (available/unavailable)

9. Security & Access Control
Role-based permissions, encrypted credentials, session management, and audit logging

Role-Based Access

  • Admin — Full system access, settings, all users
  • Manager — Team & lead management, campaigns, reports
  • Telecaller — Own leads, calls, follow-ups, campaigns
  • Report Viewer — Read-only analytics access

  • Middleware-enforced role isolation on every route
  • Policy-based authorization (Lead, User, Setting policies)
  • Telecallers can only access their own assigned data

Security Features

  • Two-factor authentication (email-based OTP)
  • Session timeout configuration
  • Admin force-logout for any user
  • Account lockout after failed login attempts
  • Encrypted storage of API credentials (SMTP, WhatsApp, TCN, Zoom)
  • Password hashing with validation rules
  • CORS policy for external API endpoints
  • Complete audit trail for all user actions

Lead Intelligence & Aging

🏷️ Lead Aging Badge
  • Visual age indicator on every lead card
  • Days since lead creation (automatically computed)
  • Color-coded: fresh, aging, stale
🔍 Duplicate Detection
  • Detect duplicates by phone or email at import
  • Merge duplicate lead records (admin)
  • Duplicate leads flagged in separate view
📌 Lead Pipeline
  • Visual Kanban board with status columns
  • Drag-and-drop or button-based status moves
  • Bulk status changes across multiple leads
